Irc-logger

Tehtävänä oli dokumentoida irclokki ohjelman asennus vaiheet.

Step 1

Laitetaan commandlinelle seuraavanlainen komento

rhc app create irclokki nodejs-0.10 --from-code

https://github.com/jvanhalen/irclokki.git


Step 2

Laitetaan commandlinelle seuraavanlaiset komennot

  • rhc cartridge list
  • rhc cartridge add mysql-5.5 -a irclokki

    • Step 3

    otetaan yhteys ssh:llä sovellukseemme komennolla

  • rhc ssh irclokki

    • Annetaan muutama komento ssh ikkunassa

    • mysql
    • DROP DATABASE if exists irclog;
    • CREATE DATABASE irclog;
    • USE irclog;
    • CREATE TABLE log (id INT PRIMARY KEY AUTO_INCREMENT, timestamp BIGINT, message TEXT);

    Lopuksi

    käynnistetään sovelluksemme uudestaan antamalla komento

  • rhc app-restart irclokki

    • tämän jälkeen sovellus pitäisi löytyä osoitteesta irclokki-("username").rhcloud.com
